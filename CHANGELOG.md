# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-17

### Added
- Initial release of BidSmart Claude Skills
- **bid-manager**: Complete workflow orchestration skill
- **bid-analysis**: Tender document analysis and extraction
- **bid-verification**: Automatic analysis report verification
- **bid-requirements**: Detailed requirements extraction
- **bid-commercial-proposal**: Commercial bid document writing
- **bid-tech-proposal**: Technical bid document writing
- **bid-mermaid-diagrams**: Diagram generation and rendering
- **bid-material-search**: Material placeholder replacement
- **bid-material-extraction**: Resource library material extraction
- **bid-assembly**: Quality check and verification report
- **bid-md2doc**: Markdown to Word document conversion
- MIT License
- Comprehensive README with installation and usage guide
- Contributing guidelines
- Support for Chinese government procurement workflows

### Technical Features
- PDF and Word document parsing
- OCR integration for scanned documents
- Python scripts for document processing
- Mermaid diagram rendering
- Automated workflow orchestration
