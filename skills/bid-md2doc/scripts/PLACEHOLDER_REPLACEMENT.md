# 占位符替换使用说明

## 📋 概述

本工具包帮助您在本地计算机上完成投标文件中材料占位符的替换工作，**无需上传任何敏感材料到系统**。

## 📦 工具包内容

您下载的文件包含：

```
📁 响应文件/
├── 📄 {项目名称}-投标文件.docx       # 带占位符的Word文档
├── 📊 占位符清单.xlsx                  # 需要替换的占位符列表
├── 🐍 replace_placeholders.py         # 替换脚本
└── 📖 占位符替换使用说明.md            # 本文件
```

## 🚀 快速开始

### 第一步：检查占位符清单

1. 打开 **占位符清单.xlsx**
2. 查看"占位符清单"工作表
3. 每一行代表一个需要替换的占位符

清单包含以下列：
- **序号**：占位符编号
- **占位符原文**：Word文档中的原始文本（如：【此处插入营业执照扫描件】）
- **材料名称**：提取的材料名称（如：营业执照）
- **本地图片路径**：🟡 **您需要填写的列**
- **位置**：占位符在文档中的位置
- **上下文**：占位符周围的文字

### 第二步：填写本地图片路径

在"本地图片路径"列中填写您电脑上对应材料图片的**完整路径**。

#### ✅ 正确的路径格式示例

**Windows系统：**
```
C:\Users\张三\Documents\投标材料\营业执照.png
C:/Users/张三/Documents/投标材料/营业执照.png
D:\公司资料\证书\ISO9001认证.jpg
```

**Linux/Mac系统：**
```
/home/zhangsan/Documents/投标材料/营业执照.png
~/Documents/投标材料/营业执照.png
```

**相对路径（文件在当前目录下）：**
```
./材料/营业执照.png
材料/营业执照.png
```

#### ❌ 常见错误

- ❌ 只写文件名：`营业执照.png` （缺少路径）
- ❌ 路径不存在：`C:\不存在的文件夹\营业执照.png`
- ❌ 文件格式错误：`营业执照.pdf` （必须是图片格式）

#### ℹ️ 重要提示

- 支持的图片格式：**PNG、JPG、JPEG**
- 如果某个占位符不需要替换，可以留空
- 路径中可以包含中文
- 确保图片文件存在且可读

### 第三步：运行替换脚本

#### 前置条件

确保您的电脑上已安装Python 3.6或更高版本。

**检查Python版本：**
```bash
python --version
# 或
python3 --version
```

**安装依赖库（首次使用需要）：**
```bash
pip install python-docx openpyxl Pillow
# 或
pip3 install python-docx openpyxl Pillow
```

#### 运行脚本

1. 打开命令行/终端
2. 切换到响应文件所在目录：
   ```bash
   cd "C:\Users\您的用户名\Downloads\响应文件"
   ```
3. 运行替换脚本：
   ```bash
   python replace_placeholders.py
   # 或
   python3 replace_placeholders.py
   ```

#### 脚本执行流程

```
1. 读取 占位符清单.xlsx
2. 检查每个图片路径是否有效
3. 在Word文档中查找占位符
4. 替换为对应的图片
5. 生成新文档：{原文件名}-已替换.docx
```

### 第四步：查看结果

替换完成后，您会看到：

```
✅ 替换完成！请查看: {项目名称}-投标文件-已替换.docx
```

**注意：**
- ✅ 原始Word文档**不会被修改**
- ✅ 新文档名称带有"**-已替换**"后缀
- ✅ 占位符已被图片替换，图片会自动调整大小

## 📊 替换统计说明

脚本运行结束后会显示统计信息：

```
📊 替换统计
============================================================
✅ 成功替换: 12 个
❌ 替换失败: 1 个
⚠️  未找到: 2 个
============================================================
```

- **成功替换**：占位符已成功替换为图片
- **替换失败**：图片路径有误或图片文件损坏
- **未找到**：Excel中填写了路径，但Word文档中没有找到对应占位符

## ❓ 常见问题

### Q1: 脚本提示"无法打开Excel文件"

**原因：** Excel文件被其他程序占用

**解决：**
1. 关闭Excel程序
2. 重新运行脚本

### Q2: 脚本提示"图片文件不存在"

**原因：** 填写的路径不正确或文件被移动/删除

**解决：**
1. 检查路径拼写是否正确
2. 确认图片文件确实存在
3. 使用绝对路径（完整路径）而非相对路径

### Q3: 替换后图片太大或太小

**说明：** 脚本会自动调整图片尺寸

- 最大宽度：15厘米（适合A4纸）
- 最大高度：20厘米
- 保持图片原始宽高比

**如需调整：**
- 方案1：在替换前调整原始图片尺寸
- 方案2：替换后在Word中手动调整

### Q4: 路径中包含中文导致错误

**解决：**
1. 保存Excel时选择"UTF-8编码"
2. 或使用英文路径名

### Q5: 部分占位符未被替换

**可能原因：**
1. Word文档中不存在该占位符（已手动删除或修改）
2. 占位符文本不完全匹配（如多了空格）
3. Excel中该行留空（故意跳过）

**解决：**
- 查看脚本输出的"未找到"列表
- 在Word中搜索占位符原文，确认是否存在
- 如需要，手动在Word中替换

### Q6: 替换后Word无法打开

**可能原因：**
1. 原始Word文档已损坏
2. 磁盘空间不足

**解决：**
1. 检查原始文档是否能正常打开
2. 确保有足够的磁盘空间（至少50MB）
3. 尝试在其他位置运行脚本

2. 运行脚本时选择要处理的文档

## 📚 附录

### 支持的图片格式详情

| 格式 | 扩展名 | 推荐 | 说明 |
|------|--------|------|------|
| PNG | .png | ✅ 推荐 | 无损压缩，适合截图和扫描件 |
| JPEG | .jpg, .jpeg | ✅ 推荐 | 有损压缩，文件较小 |
| BMP | .bmp | ⚠️ 不推荐 | 文件过大 |
| GIF | .gif | ❌ 不支持 | Word兼容性差 |
| TIFF | .tif, .tiff | ❌ 不支持 | 需要额外处理 |

### 图片优化建议

为获得最佳效果，建议：

1. **分辨率：** 300 DPI（打印级别）
2. **尺寸：** 宽度 1500-2000 像素
3. **文件大小：** 每张 < 2MB
4. **色彩模式：** RGB（彩色）或灰度
5. **格式：** PNG（证书、截图）或 JPEG（照片）
